# 🌿 Meu Portfólio

Opaa! Prazer, eu sou a Lety.

Este pequeno projeto foi feito para mostrar um pouco mais de mim e da minha trajetoria até então.

A tecnologia faz parte da minha vida desde cedo e, ao longo dos anos, ela se transformou em uma das minhas maiores formas de aprender, criar e resolver problemas. Atualmente atuo como desenvolvedora full stack, com foco em automações, integrações e desenvolvimento web, enquanto continuo explorando temas como APIs, arquitetura backend, integrações e boas práticas de desenvolvimento.

Além do código, também gosto de livros, natureza e daqueles momentos mais tranquilos que ajudam a colocar as ideias no lugar. Por isso, este portfólio foi pensado para transmitir um pouco dessa combinação entre tecnologia, aprendizado contínuo e simplicidade.

## O que você vai encontrar aqui

* Um pouco sobre mim
* Tecnologias que fazem parte da minha jornada
* Projetos que venho desenvolvendo
* Formas de entrar em contato comigo

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Executando localmente

Clone o repositório:

```bash
git clone https://github.com/lettyfranca/portfolio-letty.git
```

Abra o arquivo `index.html` no navegador ou utilize uma extensão como Live Server para uma melhor experiência durante o desenvolvimento.
